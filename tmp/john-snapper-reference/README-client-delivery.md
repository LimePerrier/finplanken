# Client delivery test package

Open `dashboard.html` to test the John Snapper dashboard.

Client files live under `client-files/john-snapper/`:

- `01-dashboard-data/` - Excel workbook used to generate or verify the dashboard data.
- `02-plan-pdf/` - Full financial plan PDF shown in the Full Report tab.
- `03-commentary-docx/` - Advisor commentary DOCX shown in the Commentary tab. A PDF preview can sit here too when browser preview is useful.

For a new client, copy this folder structure, replace the files in those three locations, and update the links in `dashboard.html` if the filenames change.
